# Note Link Quick Start Guide

## What is it?

A feature that lets you create clickable links to other notes in your workspace, similar to Wikipedia or Notion's internal links.

## Quick Start

### 1. Create a Link

While editing a note:
1. Type `/` to open commands
2. Type `note` or select "Note Link"
3. Search for and select a note
4. Done! A blue badge appears with the note link

### 2. Click a Link

Simply click on any note link badge to open that note.

## Tips

- Links show the note title, even if you rename the note later
- You can't link to the note you're currently editing
- Use search in the selector to quickly find notes
- Links work in all rich-text notes (not in drawing/mindmap notes)

## Keyboard Shortcuts

- **Slash menu**: `/` then type `note`
- **Select**: Enter or Tab
- **Cancel**: Escape
