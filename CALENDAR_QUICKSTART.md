# Calendar Quick Start Guide

## Accessing the Calendar

1. Open Notes Desktop
2. Look at the **bottom toolbar** (status bar)
3. Click the **"Calendar"** button (purple calendar icon)

## Creating Your First Event

1. Click **"New Event"** button (top right of the modal)
2. Enter an event title (e.g., "Team Meeting")
3. Optionally add a description
4. Select start and end dates
5. Choose a color for the event
6. Click **"Create Event"**

## Quick Event Creation from Calendar

1. In Calendar view, click on any date
2. The event form opens with that date pre-selected
3. Fill in the details and save

## Viewing Your Events

### Calendar View
- See all events in a monthly grid
- Events appear as colored blocks on their dates
- Navigate months using the arrow buttons
- Click "Today" to jump to current date

### List View
- Click the "List" button to switch views
- See all events in chronological order
- Each event shows title, description, date, and time

### Upcoming Events
- Check the sidebar on the right
- Shows the next 5 upcoming events
- Quick access to event details

## Editing Events

1. Click on any event (in calendar, list, or sidebar)
2. Modify the event details
3. Click **"Save Changes"**

## Deleting Events

1. Click on the event to edit it
2. Click the **"Delete"** button (bottom left)
3. Confirm the deletion

## Event Options

### All-Day Events
- Check the "All day event" box
- Time fields will be hidden
- Event spans the entire day(s)

### Color Coding
- Choose from 8 colors
- Use colors to categorize events (work, personal, urgent, etc.)

### Multi-Day Events
- Set different start and end dates
- Event will appear on all dates in the range

## Tips

- **Keyboard shortcuts**: 
  - Press `Escape` to close modals
  - Press `Enter` in the form to save
- **Real-time sync**: Events update automatically across all open windows
- **Click dates**: Clicking a date in calendar view pre-fills that date in the form
- **Event limits**: Calendar cells show up to 2 events, with "+X more" indicator

## Common Use Cases

### Meeting Scheduling
1. Create event with meeting title
2. Set exact time
3. Add description with meeting agenda/link
4. Use blue color for general meetings

### Deadline Tracking
1. Create all-day event on deadline date
2. Use red color for visibility
3. Add description with deliverables

### Personal Reminders
1. Create event with reminder description
2. Set time when you want to be reminded
3. Use purple color for personal items

### Event Series
1. Create first event
2. Duplicate and modify dates for subsequent events
3. Use same color for related events

## Troubleshooting

**Events not showing?**
- Check if you're viewing the correct month
- Try switching to List view to see all events
- Refresh the modal by closing and reopening

**Can't save event?**
- Ensure title is filled in (required)
- Check that start date is before end date
- Verify dates are valid

**Real-time updates not working?**
- Check your internet connection
- Close and reopen the calendar modal
- Refresh the entire application
